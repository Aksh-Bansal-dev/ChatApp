# ChatApp
A simple chat app using web sockets.
Mostly followed [this](https://www.youtube.com/watch?v=jD7FnbI76Hg).

## Technology used
 - Nodejs
 - Websockets
 - HTML, CSS, Vanilla Js

## How to use
- Install npm packages ```npm install```.
- Start server ```npm start```.
- Visit [website](http://localhost:5000).
